7. 无障碍系统

字号大小、对比度、文字间距和行高

无障碍功能

所有无障碍控制现在都在右下角的悬浮面板中。您可以调整：

  • 字体大小缩放
  • 对比度模式
  • 字间距
  • 行高

此文本将响应所有无障碍设置。尝试调整悬浮面板中的控件以查看更改。

设计系统变量

您可以动态调整核心设计变量：

  • 品牌色调 (0-360°)
  • 品牌饱和度 (0-100%)
  • 圆角缩放
  • 间距缩放
CSS
/* 无障碍设置 */ /* 字体大小缩放 */ [data-font-size="small"] { --font-scale: 0.875; } [data-font-size="normal"] { --font-scale: 1; } [data-font-size="large"] { --font-scale: 1.125; } [data-font-size="x-large"] { --font-scale: 1.25; } /* 对比度模式 */ [data-contrast="high"] { --color-foreground: hsl(0, 0%, 0%); --color-background: hsl(0, 0%, 100%); } [data-contrast="ultra"] { --color-foreground: #000000; --color-background: #ffffff; } /* 字间距 */ [data-letter-spacing="wide"] { letter-spacing: 0.05em; } [data-letter-spacing="wider"] { letter-spacing: 0.1em; } /* 行高 */ [data-line-height="relaxed"] { line-height: 1.8; } [data-line-height="loose"] { line-height: 2; }
