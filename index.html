♿ 无障碍功能

字体缩放

支持 5 级字体大小调整，适应不同视力需求

// 使用键盘快捷键
Ctrl/Cmd + + : 放大字体
Ctrl/Cmd + - : 缩小字体
Ctrl/Cmd + 0 : 重置大小

对比度模式

提供 3 种对比度级别，改善文字可读性

  • 标准模式：默认配色方案
  • 高对比度：增强色彩饱和度和文字粗细
  • 超高对比度：纯黑白配色，最大对比

阅读辅助

提供字间距和行高调整，提升阅读体验

  • 字间距：3 级可调，减少文字拥挤感
  • 行高：3 级可调，让阅读更轻松
  • 减少动画：关闭动画效果，避免干扰

使用方法

// 在 CSS 中启用字体缩放
html {
font-size: calc(16px * var(--font-scale));
}
// 在 HTML 中设置属性
<html data-font-size="large">
<html data-contrast="high">
<html data-reduce-motion="true">
// JavaScript API
document.documentElement.style.setProperty('--font-scale', 1.25);
document.documentElement.setAttribute('data-contrast', 'high');
